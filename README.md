# DOCKER
- up docker-compose
```sh
docker-compose up
```
- up docker-compose and rebuild the project
```sh
docker-compose up --build
```

- Stop and delete a container
```sh
docker-compose down
```

- Stop, delete the container and all images
```sh
docker-compose down --rmi all
```
